## 🎯 O que aprendi

- Como verificar integridade de arquivos de texto com `diff`
- Comparação de arquivos binários byte a byte com `cmp`
- Funcionamento de funções Hash (MD5) para garantir integridade
- Conceito de criptografia simétrica com Cifra de César via Python
- Controle de fluxo de trabalho pela linha de comando

---

## 📋 Atividades

| # | Tema | Ferramenta | Descrição |
|---|------|-----------|-----------|
| 1.1 | Integridade de texto | `diff` | Comparar dois arquivos e identificar alterações no conteúdo |
| 1.2 | Integridade binária | `cmp` | Comparar arquivos binários e localizar diferenças byte a byte |
| 1.3 | Verificação com Hash | `md5sum` | Gerar e comparar hashes MD5 para verificar integridade |
| 1.4 | Cifra de César | `Python` | Cifrar e decifrar texto com deslocamento para proteger confidencialidade |
| 1.5 | Fluxo via terminal | `CLI` | Gerenciar tarefas e fluxo de trabalho pela linha de comando |
